# 📊 Sales Data Analysis using Python


## 📖 Project Overview
This project presents an exploratory data analysis (EDA) of a retail sales dataset using Python.

The main objective is to analyze sales performance, customer behavior, regional performance, and product trends in order to generate business insights and support data-driven decision-making.

The analysis was performed using Pandas for data manipulation and Matplotlib for data visualization.


## 📂 Dataset
The dataset contains 9,800 sales records and 18 features, including:

- Order Date
- Ship Date
- Customer Segment
- Region
- Category
- Product Name
- Sales


## 🛠️ Tools & Libraries

The following Python libraries were used in this project:

- Pandas
- Matplotlib
- Jupyter Notebook


## ❓ Business Questions

The objective of this analysis is to answer the following business questions:

1. Which product category generates the highest sales?
2. Which region generates the highest sales?
3. Which customer segment contributes the most to total sales?
4. Which product categories perform best in each region?
5. Which customer segment generates the highest sales within each product category?
6. How do sales change over time?
7. Which products generate the highest revenue?


## 📊 Analysis Workflow

The project follows these main steps:

1. Data Loading
2. Data Understanding
3. Data Cleaning
4. Exploratory Data Analysis (EDA)
5. Business Insights
6. Recommendations


## 📈 Key Findings

The analysis revealed the following key insights:

- Technology is the highest-performing product category.
- The West region generates the highest sales.
- Consumer is the most valuable customer segment.
- Technology performs best across all regions.
- Overall sales show an upward trend over time.
- The Canon imageCLASS 2200 Advanced Copier is the highest-revenue product.
